# Particle Showcase

Flutter showcase project generated from the uploaded `particle_background.dart`.

## What is included

- Full-screen animated particle background
- Touch interaction (drag to repel particles)
- Landing-page style showcase layout
- Ready to adapt for portfolio, product promo, or toolkit demo

## Run

```bash
flutter pub get
flutter run -d chrome
```
